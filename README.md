# DenJOY DentWeb Automation Agent

DentWeb 수술기록을 자동으로 추출하여 DenJOY에 업로드하는 Windows 에이전트입니다.

## 설치

1. [최신 릴리즈](https://github.com/denjoy-korea/dentweb-agent/releases/latest)에서 `dentweb-agent.exe` 다운로드
2. 병원 PC에서 실행
3. DenJOY 앱 설정 화면에서 복사한 에이전트 토큰 입력

## 초기 설정

처음 실행하면 두 단계를 진행합니다:

### 1단계: 토큰 입력
DenJOY 앱 > 설정 > 덴트웹 자동화에서 토큰을 복사하여 붙여넣기

### 2단계: 클릭 좌표 학습
덴트웹 프로그램을 열고, 에이전트가 안내하는 각 단계에서
해당 버튼 위에 마우스를 올린 뒤 Enter를 누릅니다:

1. **경영/통계 메뉴 버튼** - 덴트웹 메인 화면의 통계 메뉴
2. **임플란트 수술통계 탭** - 수술통계 카테고리
3. **조회기간 설정** - 기간 선택 영역
4. **조회 버튼** - 데이터 조회 실행
5. **엑셀 다운로드 버튼** - Excel 내보내기

## 동작 방식

1. DenJOY 서버에 주기적으로 폴링 (기본 30초)
2. 실행 요청 감지 시 학습된 좌표로 덴트웹 자동 클릭
3. 다운로드된 Excel을 DenJOY 서버에 업로드

## 파일 구조

- `config.json` - 서버 URL, 토큰 등 (첫 실행 시 자동 생성)
- `dentweb_steps.json` - 클릭 좌표 (학습 모드에서 생성)
- `agent.log` - 실행 로그

## 좌표 재설정

덴트웹 UI가 변경되었거나 좌표가 맞지 않을 경우:
1. `dentweb_steps.json` 파일 삭제
2. 에이전트 재시작
3. 학습 모드가 자동 실행됨

## 요구사항

- Windows 10 이상
- DentWeb 프로그램이 설치되어 있어야 함
- 화면 해상도가 변경되면 좌표 재설정 필요
